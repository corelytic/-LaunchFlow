Corelytic LaunchFlow | Quick Start

Product
- Corelytic LaunchFlow
- Campaign Approval & Delivery Operations System

Quick Start
1. Open MainFiles/index.html directly in a modern browser, or serve the package from a local web server.
2. Use the seeded workspace immediately after opening.
3. Navigate the sidebar routes to review Dashboard, Campaigns, Launch Matrix, Revisions, Client Approval Room, LaunchBrain, Delivery Center, Timeline, Reports, and Settings / Backup.

Folder Structure
- MainFiles/
  - index.html
  - documentation.html
  - readme.txt
  - changelog.txt
  - license.txt
  - assets/
  - screenshots/
- Demo/
  - index.html
  - assets/
- PreviewAssets/
  - thumbnail.png
  - preview.png
- FINAL_RELEASE_REPORT.txt

How To Open Demo
- Open Demo/index.html directly with file:// or from a local server.
- Demo is a standalone synced copy intended for preview and testing.

How To Use MainFiles
- Open MainFiles/index.html for the buyer package.
- MainFiles includes the application, documentation, screenshots, and all release notes.

Requirements
- Modern browser with LocalStorage enabled
- No backend
- No database
- No API keys
- No build tools required

Browser Support
- Microsoft Edge
- Google Chrome
- Mozilla Firefox
- Safari (recent desktop version)

Support Notes
- Works from file:// and local server mode
- LaunchBrain is rule-based and offline
- Workspace data is stored locally in browser LocalStorage
- Importing workspace JSON overwrites the current workspace only after confirmation
